GitHub Set Up

  1. Log in to your GitHub account.
  2. Create a new repository
  3. Create a folder in c-drive and name it similar to repository name
  4. Open the folder in VS code
  5. Create a file in the folder
  6. Open command prompt in terminal: using ctrl+j
  7. Then type the following command:
    1. git init: Initialize the Git repository
    2. git add . : Added all file
    3. git commit -m "meaningful comment": Commenting the changes
    4. git branch -M main: Selecting main branch for storing files
    5. git remote add origin [address]: Git repository address.
    6. git push -u origin main: Uploading all file in repository.
